# Финансовый Аналитический Проект

Этот проект предназначен для анализа финансовых транзакций за указанный месяц. Он собирает данные о расходах, переводах, курсах валют и ценах на акции, а также генерирует подробный отчёт в формате JSON.

## Описание

Основные функции проекта:

- Получение транзакций за выбранный месяц
- Анализ расходов по категориям
- Поиск переводов между лицами
- Получение актуальных курсов валют и цен на акции
- Генерация отчёта с результатами анализа и внешними данными

## Установка

Для работы проекта необходимо установить зависимости:

```bash
pip install pandas
Также убедитесь, что все модули (api_data, services, src.reports, utils) доступны в проекте.

Использование
Запустите скрипт, передав дату в формате день.месяц.год:

bash

python main.py 20.05.2018
Это вызовет функцию main, которая выполнит сбор данных, анализ и сохранит результат в файл output.json.

Структура проекта

your_project/
├── main.py               # Основной скрипт (этот файл)
├── api_data.py           # Модуль для получения данных API (курсы валют, цены акций)
├── services.py           # Логика поиска переводов между лицами
├── src/
│   └── reports.py        # Генерация различных отчётов по транзакциям
└── utils.py              # Вспомогательные утилиты (например, получение транзакций из файла Excel)
Логирование
Логи записываются в файл app.log и выводятся в консоль. Они помогают отслеживать выполнение программы и возможные ошибки.

Пример запуска
bash

python main.py 20.05.2018
После выполнения будет создан файл output.json с полным отчётом.

Дополнительная информация
Перед запуском убедитесь, что все необходимые модули реализованы и доступны. Формат даты при вызове функции — дд.мм.гггг.
